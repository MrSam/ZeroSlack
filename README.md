# ZeroSlack

A Slack client that focuses on user experience and security.

## Development on MacOSX

1) Clone this repo locally

```bash
$ git clone https://github.com/MrSam/ZeroSlack 
```

2) Install the required dependencies 

```bash
$ npm install 
```

3) Run 

```bash
$ npm install
```
